## Dependency: Set

A variable, which is just declared before, is set a value by a method.

### Supported Patterns

```yaml
name: Set
```
#### Syntax: Set Definitions

```text
Assignment:
    var = value;
```

##### Examples

###### Method Set Local Var

```java
//// Foo.java
public class Foo {
    public static final String MSG = "MSG";
    
    public void getNum(){
        int i = 1;
        String hello = "Hello";
        hello = "World";
    }
}
```
